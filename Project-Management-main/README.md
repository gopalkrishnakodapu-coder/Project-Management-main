# 🚀 Project Management Application

A full-stack Project Management Application built using the MERN Stack. This application helps users efficiently manage projects, organize tasks, collaborate with team members, and track project progress through an intuitive and responsive interface.

---

## 📌 Features

- 🔐 User Authentication & Authorization
- 👤 User Profile Management
- 📁 Create, Update & Delete Projects
- ✅ Task Management
- 👥 Team Collaboration
- 📊 Project Progress Tracking
- 🔍 Search & Filter Projects
- 📱 Responsive User Interface

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/Project-Management.git
```

### Navigate into the project

```bash
cd Project-Management
```

### Install Frontend Dependencies

```bash
cd client
npm install
```

### Install Backend Dependencies

```bash
cd ../server
npm install
```

---

## ▶️ Running the Application

### Start Backend

```bash
cd server
npm start
```

### Start Frontend

```bash
cd client
npm start
```

The application will be available at:

```
http://localhost:3000
```
